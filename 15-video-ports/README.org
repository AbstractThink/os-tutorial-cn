/事先需要 Google 的概念： I/O ports(端口)/

*目标：学习如何使用 VGA 卡的数据端口*

我将使用 C 语言通过 =I/O= 寄存器和端口和设备进行通信。

打开 =drivers/ports.c= 文件，查看 C 语言汇编的内联语法。它有些特殊，
像是将源代码在目标位置处展开一样，这个有趣的语法会将多个赋值变量展开。

当你理解了这些概念，打开 =kernel/kernel.c= 文件，时而有一个使用的例子。

在这个例子中，我们将验证映射屏幕光标位置的 =I/O= 端口。具体地，我们用 =14= 号值查询 =0x3d4= 端口，用于请求光标高位位置，接着使用 =15= 号请求低位位置。

当查询完之后，结果存放在 =0x3d4= 端口。

不要错过使用 =gdb= 的机会，我们可以用它监视 C 变量的值。因为我们不能将它们打印到屏幕上。
因此，给具体行设置一个断点， =breakpoint kernel.c:21= , 使用 =print= 命令验证变量。

你为我们花费时间交叉编译 =gdb= 感到高兴吗？ :)

最后，我们在查询到的光标位置写入一个字符。
